# 3. 游戏设计

## 玩家跳跃能力的等级：

- 跳跃,单方向。
- 跳跃键按下时长会决定跳跃的最高点。
- 能够达到二段跳的高度，加上左右移动。
- 在空中可以通过左右移动进行细微的调整。
- 三段跳的高度，最高高度。
- 攀爬技能。
- 第三次跳跃的时候反向跳跃。
- 冲刺的使用。
- 火箭。

## 技巧

- 按下之后不松手，到达跳跃的最高点之后再马上按第二次跳跃如此往复。
- 跳跃之后，可以在空中去控制，左右。
- 第三次跳跃的时候反向跳跃

# 4. 游戏制作方法论


## 做游戏的时候

切换思维

* 技术思维
* 设计角度（策划思维）
* 美术/音乐角度

## 一个游戏的结构

* 关卡有结束
* 惩罚
  * 角色死亡
  * 血量
  * 重玩
  * 生命数量减掉 1 
* 成就感




## 游戏制作流程

1. 分析，定目标，确定三个关卡。
2. 第一章：可玩的版本 MVP(最小可验证版本)
3. 第二章: 扩充到了是个关卡，涉及一点美术的东西。



## 游戏制作的方法论

1. 先有一个可玩的版本，比较模糊的目标。
2. 不断玩。
3. 玩的过程中完善。
4. 重复 2 ~ 4。
